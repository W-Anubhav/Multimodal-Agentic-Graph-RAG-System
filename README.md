# Multimodal-Agentic-Graph-RAG-System
A Multimodal Agentic Graph-RAG pipeline. Uses Computer Vision (PyTorch/OCR) to ingest complex financial PDFs, a dual-database architecture (Neo4j &amp; Qdrant) to map spatial layout and semantic meaning, and a LangGraph ReAct agent to autonomously route queries and prevent LLM hallucinations.
